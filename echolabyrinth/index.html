Echo Labyrinth

Audio-only maze navigation using footsteps, wall thuds, ping echoes, and a directional beacon.

Controls: Arrow keys move, Space pings, Enter repeats instructions, Esc returns to menu.
Audio: Footstep = move, thud = wall, warning = hazard, bright chime = exit.
Goal: Track beacon pan/volume cues and reach the goal in the fewest moves.
