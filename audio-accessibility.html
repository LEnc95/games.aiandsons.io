Quick Start

Echo Labyrinth, Beat Rail, and Branching Audio Adventure are tagged: 100% playable without sight; keyboard and screen reader friendly.

These recommendations reflect common interactive-fiction and accessibility-community practices for keyboard-first play and clear speech output.

  • Use a stable pairing such as NVDA + Firefox, JAWS + Chrome, or VoiceOver + Safari.
  • Keep your screen reader in focus/forms mode while playing so game keybinds pass through.
  • Set speaking rate to a comfortable speed and reduce extra punctuation/verbosity for faster navigation.
  • If game narration and screen reader speech overlap, pause or mute one voice source for cleaner audio.

Keybindings By Game

Game Core keys What they do
Echo Labyrinth Arrow keys Space Enter Esc Move in four directions, ping for echoes, repeat instructions, or return to menu.
Beat Rail Left Down Right Space R Hit rhythm lanes, pause/resume and replay instructions, and restart the song.
Branching Audio Adventure 1-9 Up/Down Enter R T V Q Choose story options by number, navigate choices, repeat narration, stop voice, restart, or quit from endings.

Troubleshooting Tips

  • If key presses do not register, click once inside the page to return focus, then continue with keyboard input.
  • If speech sounds clipped, lower speech rate slightly and reduce competing browser tab audio.
  • If the game and screen reader both read everything, keep screen reader speech for navigation and use game narration only on-demand.

After changing screen-reader settings, reload the game so focus and live-region announcements start in a clean state.
