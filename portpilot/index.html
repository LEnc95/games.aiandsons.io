Port Pilot

Route each cargo pod to the dock with the same color. Rotate nodes before pods reach the edge.

Desktop: move cursor with WASD or arrows, Space or Enter rotates selected node. P pause, R restart, F fullscreen.

Mobile: use D-pad to move cursor and the Rotate button to turn selected nodes.
